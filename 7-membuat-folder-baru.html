FOLDERBARU (new folder)

adalah sebuah tempat penyimpanan yang digunakan untuk mengorganisir file dan data di komputer. Folder bisa diibaratkan seperti sebuah laci atau map di dunia nyata yang digunakan untuk menyimpan dokumen atau barang-barang tertentu. Dengan menggunakan folder, pengguna dapat mengelompokkan file berdasarkan kategori atau tujuan tertentu, sehingga memudahkan untuk menemukan dan mengelola file tersebut.
